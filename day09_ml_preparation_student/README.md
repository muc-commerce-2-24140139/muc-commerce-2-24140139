## 用自己的话回答

- 什么是特征，什么是标签：特征是用于描述样本的、可测量的属性或特性；标签是我们希望模型预测的目标变量。
- 为什么要保留测试集：测试集用于评估模型在未见过的数据上的表现，从而估计模型的泛化能力，防止过拟合。
- 为什么83%准确率仍可能没有用：因为当数据集中类别不平衡时（如95%未流失，5%流失），一个总是预测“未流失”的模型也能达到95%的准确率，但它一个流失用户都找不出来，在业务上毫无价值。
